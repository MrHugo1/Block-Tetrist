🧪 Cách Test

Test Mode Selection:

  1. Mở game - Mode selector phải hiển thị
  2. Chọn từng mode (Basic, Blocker, Multiplier, Blast)
  3. Kiểm tra game bắt đầu với mode đã chọn
  4. Kiểm tra high scores được lưu riêng biệt

Test Drag & Drop:

  1. Chọn piece từ pool
  2. Kéo lên bàn chơi
  3. Kiểm tra drop preview
  4. Thả để đặt piece
  5. Test drop ở vị trí không hợp lệ

📱 Mobile vs Desktop

Desktop:

  • Click để chọn piece
  • Drag & drop để đặt
  • Right-click để bỏ chọn

Mobile:

  • Touch để bắt đầu drag
  • Drag & drop để đặt
  • Hold/Swap buttons
